If you're under attack, it's highly recommended to enable the next setting, so DragonEye will disable all the connections to your server database.


Because the database connection will be disabled, you will not be able to access the administration panel anymore, so you must do those things manually.

- To disable under attack mode, you must follow this guide:
          https://dragoneyecms.com/under_attack.php?guide=0

- To add/edit/delete content pages, you must use this online tool
          https://dragoneyecms.com/under_attack.php?content
