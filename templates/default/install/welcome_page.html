Please take 5 minutes of your time in order to install DragonEye CMS.

It's simple, you just have to complete database configurations, database structure (needed for server compatibility), server name, website url, meta tags, access levels and mail connection.

After, you can access the administration panel with your account to manage the whole website (admin panel account it's the same as in-game admin account, so if you don't have one just use register then manually change it's access level from database).
